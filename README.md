# Sistema Integral de Gestión de Pasantías (SIGP)

**Proyecto de Estancia Profesional - TSU Desarrollo de Software**
**Grupo 23**

## Descripción
Plataforma web para gestionar el ciclo completo de pasantías, conectando estudiantes con empresas a través de la Cámara de Comercio. Incluye módulos de seguridad, gestión de perfiles, plazas, postulaciones y control de calidad ISO 9001.

## Documentación
La documentación técnica se encuentra en la carpeta `/docs`:
- `reporte_proyecto.html`: Especificación de Requerimientos (SRS) completa.
- `visualizador_bd.html`: Mapa interactivo de la base de datos.
- `plan_maestro_pruebas.md`: Estrategia de QA.

## Instalación
1. Clonar repositorio en `htdocs`.
2. Importar `public/databases/sigp_complete.sql`.
3. Configurar `.env` con credenciales de BD.
