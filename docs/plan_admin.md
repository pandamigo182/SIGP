# Plan de Documentación Administrativa y Trello

## Objetivo
Generar el contenido detallado para los formatos administrativos solicitados ("Plan de Trabajo" y "Especificaciones de Requerimientos") y diseñar el storyboard de Trello, adaptados al proyecto SIGP.

## Limitaciones
- No es posible editar archivos binarios (.xlsx, .docx) directamente.
- No es posible leer texto de imágenes (integrantes).
**Solución**: Se generarán archivos Markdown con el contenido estructurado listo para copiar/pegar, usando "Edwin Juárez" y placeholders para los demás integrantes.
