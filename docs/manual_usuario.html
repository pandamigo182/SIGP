Manual de Usuario - SIGP

Bienvenido al Sistema Integral de Gestión de Pasantías.

Este documento le guiará en el uso de las funciones principales.

1. Acceso al Sistema

Ingrese a la URL proporcionada. Verá la pantalla de Login.

  • Estudiantes: Usar correo personal.
  • Empresas: Usar correo corporativo.

2. Dashboard Estudiante

Al ingresar, encontrará:

  • Mis Postulaciones: Estado de sus solicitudes.
  • Perfil: Donde puede subir su CV en PDF.
  • Ofertas: Listado de plazas disponibles.
